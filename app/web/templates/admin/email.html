메일 설정

UUID 형식. #} {# Application (client) ID — Azure AD app registration 의 client ID. UUID 형식. #} {# Client secret — masked input + 변경 토글. default 상태: 비활성화 + placeholder 에 마지막 4 자 mask 표시.
